Overview

Google Tasks Multirow is a Chrome extension designed to improve the visual layout and usability of Google Tasks by applying custom styling to task containers and task cards. It enhances the user interface by enabling multirow layouts for task lists without collecting or transmitting any user data.

What Data We Collect

This extension does not collect, store, or transmit any personal or sensitive user data. All operations are performed entirely in your browser and are never sent to any external server.

Permissions Explanation

This extension requires host permissions for https://tasks.google.com/* to apply layout and style adjustments. These permissions are used only to modify the visual structure of the Google Tasks interface. The extension does not access the contents of your tasks or any other personal data.

Third-Party Services

Google Tasks Multirow does not use any third-party services, analytics tools, or external APIs.

Data Storage

No data is stored by the extension, either locally or remotely.

Changes to This Policy

If there are future updates that involve changes in data access or use, this policy will be updated accordingly and users will be notified via the Chrome Web Store listing.
